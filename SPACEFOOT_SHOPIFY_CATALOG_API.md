# Spacefoot Shopify Catalog API Notes

The feed is generated by GitHub Actions with Shopify Admin GraphQL API credentials stored in GitHub Secrets.

Required identifiers:

- `SHOPIFY_CATALOG_ID` / `SHOPIFY_CATALOG_GID` — the Spacefoot / France EUR Shopify catalog.
- `SHOPIFY_PUBLICATION_GID` — the catalog publication used to read included products.
- `SHOPIFY_PRICE_LIST_GID` — the EUR price list attached to the catalog.

The workflow intentionally does not use Storefront API and does not expose the Shopify Admin access token to GitHub Pages. Only generated static files from `public/` are published.
